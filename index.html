Computer Topology

The computer topology refers to schematic description of various network components such as node, links, peripherals and so on.

The various topologies are-
  1. Point-Point Topology

    • Simple and most convinient
    • 2 nodes are directly connected
    • Faster and more reliable
    • Used for small areas

  2. Bus Topology

    • Local area network in which all the nodes are connected to a single cable
    • The cable to which all the nodes are connected is called backbone
    • If one computer system breaks it will not affect other systems
    • Easy and inexpensive
    • In this network one of the computer act as a server

  3. Ring Topology

    • Nodes are connected in the form of a ring
    • Its data flow is uni-directional
    • High bandwidth
    • can cover a large distance

  4. Star Topology

    • All the devices are connected to a single device which act as a hub or switch
    • If a connection fails for a system it can be replaced easily
    • Adding or removing system is possible without distruption
    • It require more connection
    • If the central device fails the entire network fails

  5. Tree Topology

    • Topology which is having a tree structure in which all the computers are connected like the branches connected to the tree
    • Tree network topology is considered to be the simplest topology
    • Also called as Star-Bus topology
    • Expansion of network is easy

  6. Mesh Topology

    • Computers and network devices is inter-connected with one another
    • Setup allows for most transition to be distributed even if the connection goes down
    • Also known as meshnet
    • It is a topology commonly used for wireless network
    • Can manage high data traffic
